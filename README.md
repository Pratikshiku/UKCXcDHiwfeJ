### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 忘记密码
4. 首页
5. 商品信息
6. 公告信息
7. 在线客服
8. 购物车
9. 个人中心
10. 我的订单
11. 我的地址
12. 我的收藏


**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息）
3. 用户管理
4. 商品分类管理
5. 商品信息管理
6. 系统管理（关于我们、轮播图管理、系统简介、公告信息、在线客服）
7. 订单管理
